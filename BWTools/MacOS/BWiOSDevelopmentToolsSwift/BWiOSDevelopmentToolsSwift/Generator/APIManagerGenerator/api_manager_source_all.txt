Template Code:
    URL: <#Back End Interface#> Name: <#Comment#> APIManager: <#APIManager Name#> Interface: <#INTERFACE_NAME#>

Sample Code:
    URL: washMall/appointment/getExpressCompanyList Name: 获取快递公司列表 APIManager: GetExpressCompanyList Interface: GET_EXPRESS_COMPANY_LIST
参数解释
    URL: 接口路径
    Name: 接口注释名称
    APIManager: 生成APIManager的文件名
    Interface: 宏定义接口名
